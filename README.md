
# Golden Fork Restaurant Website

A modern, responsive restaurant landing page built to showcase menu offerings, services, and reservations with a clean and conversion-focused design.

**Live Demo:** [https://forkrestaurant.vercel.app/](https://forkrestaurant.vercel.app/)
**Repository:** [https://github.com/hamnahA/goldenfork](https://github.com/hamnahA/goldenfork)

---

## Overview

This project is a front-end implementation of a restaurant website designed to provide a strong digital presence for food businesses. The layout focuses on clarity, usability, and structured content presentation to improve user engagement.

---

## Key Features

* Responsive design across all devices
* Clean and structured menu layout
* Table reservation section
* Service highlights (dine-in, delivery, takeaway)
* Testimonials section
* Consistent UI with modern design principles

## Getting Started

Clone the repository:

```bash
git clone https://github.com/hamnahA/goldenfork.git
cd goldenfork
```

Run locally:

```bash
open index.html
```

Or use a development server such as Live Server in VS Code.

---

## Customization

* Update content in `index.html`
* Modify styles in `style.css`
* Edit logic in `script.js` or TypeScript files
* Replace assets in `/images`
* Add screenshots in `/screenshots`
